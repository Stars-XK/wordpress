# GodMainCode WordPress Theme

现代化、扁平化设计的WordPress主题，支持毛玻璃效果、响应式布局和丰富的功能模块。

## 主题特色

- 🎨 **扁平化设计**：简洁现代的UI风格
- 🔍 **毛玻璃效果**：优雅的玻璃质感界面
- 📱 **响应式布局**：完美适配桌面端和移动端
- 🌤️ **天气插件**：集成OpenWeatherMap API
- 🎵 **音乐播放器**：带播放列表的音乐播放器
- 📰 **新闻板块**：突出显示新闻内容
- 📝 **文章列表**：展示博客文章

## 安装步骤

### 1. 上传主题

1. 登录WordPress后台
2. 进入 **外观 → 主题**
3. 点击 **添加新主题 → 上传主题**
4. 选择 `GodMainCode.zip` 并上传
5. 点击 **激活**

### 2. 配置菜单

**创建主导航菜单**

1. 进入 **外观 → 菜单**
2. 点击 **创建新菜单**
3. 输入菜单名称：`Primary Menu`
4. 在左侧选择页面、分类或自定义链接添加到菜单
5. 在"菜单位置"中勾选 **Primary Menu**
6. 点击 **保存菜单**

**创建页脚菜单**

1. 再次点击 **创建新菜单**
2. 输入菜单名称：`Footer Menu`
3. 添加页脚链接（如：关于我们、联系我们等）
4. 在"菜单位置"中勾选 **Footer Menu**
5. 点击 **保存菜单**

### 3. 配置天气插件

1. 进入 **外观 → 自定义**
2. 点击 **Weather Settings**
3. 输入你的OpenWeatherMap API Key
   - 从 https://openweathermap.org/ 免费获取
4. 设置默认城市（如：Beijing, Shanghai等）
5. 点击 **发布**

### 4. 创建内容

**创建文章分类**

1. 进入 **文章 → 分类**
2. 新建分类：`News`（新闻）
3. 可选创建其他分类

**添加文章**

1. 进入 **文章 → 写文章**
2. 输入文章标题和内容
3. 在右侧设置"特色图片"（文章缩略图）
4. 选择文章分类（如News用于新闻板块）
5. 点击 **发布**

## 功能模块说明

### 导航菜单

- 桌面端：顶部水平导航
- 移动端：汉堡菜单（点击展开）
- 搜索按钮：右上角搜索图标，点击打开全屏搜索

### 首屏区域 (Hero)

- 显示网站标题和描述
- 统计卡片展示文章数量
- 可在自定义器中配置

### 文章列表

- 显示最新6篇文章
- 每篇文章显示：缩略图、分类、标题、摘要、作者、日期
- 点击"查看更多"跳转到文章归档

### 新闻板块

- 主新闻：显示最新1篇News分类的文章
- 新闻列表：显示接下来5篇News分类的文章
- 需要先创建News分类并添加文章

### 天气插件

- 显示当前天气图标
- 温度、湿度、风速、气压
- 需要配置API Key才能使用

### 音乐播放器

- 播放/暂停控制
- 上一首/下一首
- 进度条拖动
- 音量控制
- 播放列表选择
- 当前是模拟演示，可扩展为真实音频播放

## 自定义选项

进入 **外观 → 自定义** 可以配置：

- 站点身份：标题、标语、Logo
- Weather Settings：天气API和城市
- 菜单：导航菜单设置
- 首页设置：首页显示内容

## 开发说明

### 修改配色

编辑 `style.css` 顶部的 `:root` 变量：

```css
:root {
    --primary-color: #6366f1;    /* 主色调 */
    --secondary-color: #06b6d4; /* 次要色 */
    --accent-color: #f472b6;    /* 强调色 */
}
```

### 扩展音乐播放器

当前是模拟播放，可以在 `js/main.js` 的 `initMusicPlayer()` 函数中扩展：

- 集成真实音频文件
- 添加音频加载功能
- 实现真实播放进度

### 添加更多模块

在 `template-parts/content/` 中创建新的模板文件，然后在 `index.php` 中引用。
